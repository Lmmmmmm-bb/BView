# BViewer

使用 Chrome 扩展程序来管理你的 Bilibili 关注列表。

## 特性

- [x] 获取 Bilibili 视频封面
- [x] 获取 Up 主页背景图
- [x] 通过 UID 关注 Up
- [x] 查看 Up 的视频列表
- [x] 鼠标视频预览
- [x] 封面和背景图嵌入页面
- [ ] 鼠标视频弹幕预览
- [ ] 支持右键获取封面和背景图
- [ ] ...
